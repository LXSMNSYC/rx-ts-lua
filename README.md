# rx-ts-lua
ReactiveX for TypeScriptLua

## Install

Yarn
```bash
yarn add rx-ts-lua
```

NPM
```bash
npm i rx-ts-lua
```

## Usage

### Loading module
Note: I recommend using the `node-resolve-lua` package for loading Lua modules from Node-installed modules.

```lua
local rx = require('rx-ts-lua')
```
